local demo

Run Veldra on regtest

This demo starts bitcoind regtest plus the template manager and pool verifier. It produces a live dashboard with verdicts and reason codes.

Quick start

Clone the main repo and run the dev script. Your dashboard should begin showing verdicts within seconds.

git clone https://github.com/LeavesJ/veldra.git
cd veldra
cp .env.example .env
./scripts/dev-regtest.sh
If ports are busy, kill prior runs or change ports in your env. Keep the demo local.

What to open

Open the verifier dashboard after starting the stack.

http://127.0.0.1:8080
  • Verdict stream updates live
  • Aggregates by reason and tier
  • Active policy and thresholds
